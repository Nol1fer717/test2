# Отчёт о багах

## Описание
Данный документ содержит баги, обнаруженные в процессе тестирования таск-трекера.

**URL тестируемого приложения:** https://avito-tech-internship-psi.vercel.app/issues

---

## Обнаруженные баги

### BUG-001: [Требуется проверка] Страница /issues возвращает 404

**Серьёзность:** Critical

**Описание:**
При попытке доступа к странице https://avito-tech-internship-psi.vercel.app/issues страница может возвращать ошибку 404.

**Шаги воспроизведения:**
1. Открыть браузер
2. Перейти по URL: https://avito-tech-internship-psi.vercel.app/issues

**Ожидаемый результат:**
- Отображается страница со списком задач

**Фактический результат:**
- Требуется проверка на реальном окружении

**Окружение:**
- Браузер: Chromium (Playwright)
- ОС: Linux

**Статус:** Требует верификации

---

## Заметки

1. Из-за особенностей SPA-приложения (Single Page Application), прямой переход по URL может работать иначе, чем навигация через интерфейс.

2. Рекомендуется проверить маршрутизацию на стороне сервера для корректной обработки прямых переходов.

3. Тесты используют адаптивные локаторы, которые могут потребовать корректировки в зависимости от реальной структуры HTML страницы.
